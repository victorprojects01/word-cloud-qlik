# Qlik Word Cloud Extension

---

🇺🇸 **English**

#### Dynamic Word Cloud extension for Qlik Cloud.

Features:

Automatic word splitting
Word frequency analysis
Qlik selection on click
Tooltip with occurrences
Mouse wheel zoom
Responsive layout
Custom color palettes
Stop words filtering

Ideal for:

Comments
Surveys
SAC
Tickets
Social media analysis

---

🇧🇷 **Português**

#### Extensão de Nuvem de Palavras dinâmica para Qlik Cloud.

**Recursos**:
